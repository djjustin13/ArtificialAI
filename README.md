# Ranking app

## Project setup
```
npm install
cd server
npm install
```

### Compiles and hot-reloads for front-end development
```
npm start
```

### Compiles frond-end and minifies for production
```
npm run build
```

#### Seperate services
If you want to run all the serverices in seperate terminalls to work on the backend or see loggs
```
mongod
node server/app.js
npm run serve
```
